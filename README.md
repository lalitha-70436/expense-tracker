# 💰 Expense Tracker

A clean, responsive expense tracker web app built with vanilla HTML, CSS, and JavaScript.

## Features

- ➕ Add income and expense transactions
- 🗑️ Delete transactions
- 📊 Live balance, income & expense summary
- 💾 Persists data with `localStorage`
- 📱 Fully responsive (mobile-friendly)

## Getting Started

No installation needed — just open `index.html` in your browser.

## Usage

- Enter a **description** and **amount**
- Use a **positive** number for income (e.g. `500`)
- Use a **negative** number for expenses (e.g. `-50`)
- Hover over a transaction to reveal the **delete** button

## Tech Stack

- HTML5
- CSS3 (Grid, Flexbox, animations)
- Vanilla JavaScript (ES6+)
- Google Fonts (Poppins)
